# MultiMedia Annotator Frontend

React frontend for the [TextAnnotator](https://github.com/texttechnologylab/TextAnnotator) backend, built for creating and managing text/multimedia annotation projects.

## Features

- Project-based annotation workflow with role-based access
- Real-time collaboration via WebSockets
- AI-assisted annotation with the RAG Bot
- Image and text annotation support
- Admin panel with export and progress tracking
- Dark mode support

## Quick Start

```bash
npm install
npm run dev
```

See [Installation](getting-started/installation.md) for full setup instructions.
